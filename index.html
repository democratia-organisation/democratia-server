Documentation de l'API REST - Democratia

Utilisez l'URL de base suivie des paramètres request et parameters.

Structure des requêtes

URL : rest.php?request=NOM_ACTION¶meters=["VAL1", "VAL2"]

  • request : Nom de l'action définie dans l'API.
  • parameters : Tableau JSON encodé contenant les valeurs dans l'ordre de la procédure.

Exemple : rest.php?request=infoInternaute¶meters=["3"]

🔐 Requêtes SQL Personnalisées

Règle de sécurité : Le mot-clé SQL doit correspondre à la méthode HTTP. Toutes les requêtes personnalisées doivent obligatoirement utiliser des paramètres préparés (?).
Verbe HTTPMot-clé SQL requisExemple de request
