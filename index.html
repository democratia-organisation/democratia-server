Documentation de l'API REST - Democratia

Structure des requêtes

FORMAT : `/path/:parameters`

Exemple : /users/7

GET Récupération de données

`/users/:id_internaute/groupes` : Liste des groupes d'un utilisateur.
Pos Paramètre Type
1 id_internaute INT
/users/:courriel : Informations de profil d'un internaute.
Pos Paramètre Type
1 courriel VARCHAR
rechercherInternaute : Recherche par nom et prénom.
Pos Paramètre Type
1 nom VARCHAR
2 prenom VARCHAR
budgetGroupe : Budget utilisé par un groupe.
Pos Paramètre Type
1 id_groupe INT
budgetTheme : Budget utilisé par thématique dans un groupe.
Pos Paramètre Type
1 id_thematique INT
2 id_groupe INT
votesEffectues : Historique des votes.
Pos Paramètre Type
1 id_vote INT
2 id_groupe INT
obtenirImage : Obtenir une image d'un groupe
Pos Paramètre Type
1 id_groupe INT

POST Création d'éléments

CreerUtilisateur : Inscription d'un nouvel utilisateur.
Pos Paramètre Type
1 nom VARCHAR
2 prenom VARCHAR
3 adr_postal VARCHAR
4 mail VARCHAR
5 hashageMDP VARCHAR
CreerGroupe : Création d'un groupe de démocratie.
Pos Paramètre Type
1 nom VARCHAR
2 couleur VARCHAR
3 image VARCHAR
4 budget INT
5 nbj_dft_vote INT
6 nbj_dft_discuss INT
ajouterProposition : Soumettre une idée.
Pos Paramètre Type
1 titre VARCHAR
2 id_thematique INT
3 description TEXT
4 budget INT
5 id_groupe INT
publierImage : Publier une image d'un groupe
Pos Paramètre Type
1 id_groupe INT

PATCH Mise à jour et Actions

ModifInfoInternaute : Modifier le profil utilisateur.
Pos Paramètre Type
1 id_internaute INT
2 nom_internaute VARCHAR
3 prenom_internaute VARCHAR
4 addresse_postale VARCHAR
5 courriel VARCHAR
6 hashageMDP VARCHAR
SignalerCommentaire : Signaler un abus.
Pos Paramètre Type
1 id_commentaire INT

DELETE Suppression

SupprimerInternaute : Supprimer un utilisateur.
Pos Paramètre Type
1 id_internaute INT
