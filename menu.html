Documentation de l'API REST

Bienvenue dans la documentation de votre API REST. Voici les détails pour interagir avec l'API :

Obtenir une ressource

Utilisez le lien suivant pour obtenir une ressource spécifique :

?requete=<requête>

Exemple : ?requete=SELECT * FROM internaute

Passer des paramètres

Si la requête nécessite des paramètres, ajoutez-les sous la forme d'un tableau JSON :

?parameters=[<différents paramètres>]

Exemple : ?requete=SELECT * FROM internaute WHERE id_internaute = ?&parameters=[1]

Note : Si aucun paramètre n'est requis, il n'est pas nécessaire d'ajouter ?parameters.

Obtenir les actions par défaut disponibles

Pour obtenir la liste des actions par défaut disponibles dans l'API, utilisez la requête suivante :

?requete=getMethode

Cela retournera la documentation complète des méthodes disponibles sans nécessiter de paramètres supplémentaires.

Documentation des fonctions disponibles

Voici la liste des fonctions disponibles avec leur résumé :

  • modifNotif - Modifier les paramètres de notification d'un utilisateur.
  • ModifInfoInternaute - Modifier les informations d'un internaute dans la base de données.
  • Signalercommentaire - Signaler un commentaire problématique.
  • signalergroupe - Signaler un groupe problématique.
  • signalerproposition - Signaler une proposition problématique.
  • enregistrerCommentaire - Enregistrer un commentaire fait par un utilisateur.
  • enregistrerVote - Enregistrer un vote effectué par un utilisateur.
  • enregistrerChoixVote - Enregistrer le choix d'un utilisateur lors d'un vote.
  • demanderVote - Demander des informations sur un vote spécifique.
  • budgetGroupe - Obtenir le budget utilisé par un groupe.
  • budgetThee - Obtenir le budget utilisé par un thème spécifique.
  • groupeUtilisatuer - Obtenir la liste des groupes d'un utilisateur.
  • infoInternaute - Obtenir les informations personnelles d'un internaute.
  • budgetProposition - Obtenir le budget associé à une proposition.
  • rechercherInternaute - Rechercher des internautes en fonction de critères spécifiques.
  • votesEffectues - Obtenir la liste des votes effectués par un utilisateur.
  • sumThemesGroupe - Obtenir la somme des thèmes utilisés par un groupe.
  • supprimergroupe - Supprimer un groupe spécifique de la base de données.
  • supprimerproposition - Supprimer une proposition de la base de données.
  • Supprimerinternaute - Supprimer un internaute de la base de données.
  • supprimercommentaire - Supprimer un commentaire de la base de données.
  • ajouterProposition - Ajouter une nouvelle proposition à la base de données.
  • AjouterMembre - Ajouter un membre à un groupe existant.
  • CreerGroupe - Créer un nouveau groupe avec des paramètres spécifiques.
  • CreerUtilisateur - Créer un nouvel utilisateur avec ses informations de connexion.
  • demarrerVote - Démarrer un nouveau vote avec des paramètres définis.
  • reagirComm - Réagir à un commentaire spécifique.
  • reagircommentaire - Réagir à un commentaire en utilisant une fonction spécifique.
  • reagirProp - Réagir à une proposition spécifique.
